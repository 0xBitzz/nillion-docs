NillionClient

NillionClient(user_key, node_key, bootnodes, payments_config)

The Nillion Client

This is the main interface with the Nillion network. The Nillion Client provides APIs that you can use for:

  • secure computing in Nillion,
  • storing programs,
  • managing secrets, and
  • managing permissions
